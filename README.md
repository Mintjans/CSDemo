# CSDemo
Projeto utilizando a linguagem Go e a biblioteca demoinfocs (https://github.com/markus-wa/demoinfocs-golang/tree/master) para extrair informações de uma demo de cs e armazenar em csv, para criar um mapa de calor em R.
